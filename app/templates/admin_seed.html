Seed country news

So every country has at least one item on the map, the app uses app/data/country_news/seed.json (keyed by ISO country code). You can edit that file by hand or generate placeholders with the script.

  • Edit by hand: Open app/data/country_news/seed.json. Each key is an ISO 3166-1 alpha-2 code (e.g. US, FR). The value is an array of items with title, summary, source_url, date, source, optional lat/lon, and optional topic (geopolitics, economics, conflicts, sports).
  • Generate placeholders: Run python3 scripts/seed_country_news.py from the project root. This adds one generic entry per country for any country that has none; existing entries are left as-is.
  • Centroids: If you omit lat/lon in a seed item, the app uses app/data/country_news/centroids.json (ISO2 → [lat, lon]) for that country.
  • After editing: Replace seed.json in app/data/country_news/, then either restart the app or so the next country click loads the new data.
